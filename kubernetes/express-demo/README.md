# Express TypeScript Backend

This project is a basic Express backend using TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build TypeScript:
   ```bash
   npm run build
   ```
3. Start server:
   ```bash
   npm start
   ```

## Scripts
- `npm run build`: Compile TypeScript
- `npm start`: Run compiled server

## Structure
- `src/`: TypeScript source files
- `dist/`: Compiled JavaScript

## Requirements
- Node.js
- npm

## Development
Edit files in `src/` and rebuild to see changes.
